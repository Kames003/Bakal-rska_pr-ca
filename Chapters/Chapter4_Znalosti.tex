\chapter[Teoretické a praktické znalosti uplatněné v průběhu praxe]{Teoretické a praktické znalosti a dovednosti získané v průběhu studia uplatněné studentem v průběhu odborné praxe}
\label{sec:Znalosti}

Počas odbornej praxe som mal príležitosť aplikovať široké spektrum znalostí získaných počas štúdia na Fakulte elektrotechniky a~informatiky. Základy z~predmetov Algoritmy a~Objektovo orientované programovanie tvorili fundament pre algoritmické myslenie a~prácu s~dátovými štruktúrami. V~nasledujúcich sekciách popisujem konkrétne oblasti, ktoré mi najviac pomohli pri riešení zadaných úloh.

\section{Softvérové inžinierstvo}
\label{sec:ZnalostSWI}

Znalosti z~predmetov Softvérové inžinierstvo a~Vybrané partie softvérového inžinierstva v~kombinácii s~mojimi pracovnými skúsenosťami z~predošlých zamestnaní mi umožnili navrhnúť architektúru systému s~dôrazom na modularitu a~škálovateľnosť. Rozdelenie na vrstvy (prezentačná, aplikačná, dátová) vychádzalo z~teoretických znalostí o~viacvrstvových architektúrach a~návrhových vzoroch.

\section{Databázové systémy}
\label{sec:ZnalostDB}

Znalosti z~predmetov Databázové systémy~I a~II spolu s~predmetom Technologie databázových systémů tvorili základ pre návrh a~implementáciu databázovej vrstvy projektu. Aplikoval som princípy normalizácie pri návrhu schémy, definoval cudzie kľúče pre zabezpečenie referenčnej integrity a~vytvoril indexy pre optimalizáciu výkonu dotazov. Praktické skúsenosti s~SQL som využil pri písaní komplexných dotazov a~triggerov pre automatické generovanie geometrických hodnôt z~GPS súradníc.

\section{Vývoj webových a~mobilných aplikácií}
\label{sec:ZnalostWeb}

Pri vývoji frontend a~backend časti aplikácie som využil znalosti z~predmetov Vývoj informačných systémov a~Tvorba mobilných aplikácií. Pochopenie komponentového prístupu a~životného cyklu aplikácií mi uľahčilo prácu s~React frameworkom. Predmet Užívateľské rozhranie mi poskytol základy pre návrh intuitívneho a~responzívneho dizajnu aplikácie. Koncept REST API som aplikoval pri návrhu backend endpointov -- dodržiaval som princípy bezstavovosti a~správne používanie HTTP metód a~status kódov.

\section{Skriptovacie jazyky}
\label{sec:ZnalostPython}

Python, s~ktorým som sa zoznámil v~predmete Skriptovacie jazyky, sa stal hlavným jazykom pre backend vývoj. Skúsenosti s~parsovaním dátových štruktúr mi pomohli pri implementácii parsera pre spracovanie payloadov z~GPS trackerov. Objektovo-orientovaný prístup som aplikoval pri štruktúrovaní kódu do tried pre MQTT klienta, parser a~databázovú vrstvu.

\section{Počítačové siete}
\label{sec:ZnalostSite}

Teoretické znalosti z~predmetu Počítačové siete som využil pri práci s~komunikačnými protokolmi. Pochopenie vrstiev TCP/IP modelu mi pomohlo pri debugovaní problémov s~MQTT pripojením a~konfiguráciou TLS šifrovania. Pri implementácii WebSocket real-time komunikácie som využil znalosti o~persistent connections a~rozdieloch medzi TCP a~UDP.
