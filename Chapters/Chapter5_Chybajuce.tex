\chapter{Znalosti či dovednosti scházející studentovi v průběhu odborné praxe}
\label{sec:Chybejici}

% Rozsah: 1 strana

Počas odbornej praxe som sa stretol s~niekoľkými technológiami a~nástrojmi, ktoré neboli súčasťou vysokoškolského štúdia. Chýbajúce znalosti sa dajú rozdeliť do dvoch kategórií -- technológie, ktoré som sa naučil počas úvodných onboarding úloh, a~špecializované nástroje potrebné pre hlavný projekt.

\section{Onboarding -- DevOps technológie}
\label{sec:Onboarding}

Pred zahájením hlavného projektu som dostal sériu úvodných úloh zameraných na DevOps technológie, ktoré firma používa. Tieto oblasti neboli súčasťou bakalárskeho štúdia:

\begin{itemize}
    \item \textbf{Docker a~Docker Compose} -- Kontajnerizácia aplikácií a~orchestrácia viacerých služieb. Vytvoril som jednoduchú aplikáciu s~PostgreSQL databázou a~Nginx webserverom, všetko prepojené cez Docker Compose.
    \item \textbf{GitLab CI/CD} -- Automatizované buildy a~deployment. Nastavil som pipeline pre zostavenie Docker image a~nahranie do GitLab Container Registry.
    \item \textbf{Dokku} -- PaaS platforma pre deployment aplikácií. Nasadil som testovaciu aplikáciu na virtuálnom stroji s~Debian OS.
\end{itemize}

Tieto úlohy mi poskytli praktický základ, ktorý som následne využil pri kontajnerizácii hlavného projektu.

\section{Databázové rozšírenia}
\label{sec:ChybDatabazy}

Rozšírenia PostgreSQL -- TimescaleDB~\cite{timescaledb_docs} pre prácu s~časovými radami a~PostGIS~\cite{postgis_docs} pre geopriestorové operácie -- neboli súčasťou predmetov Databázové systémy. Musel som sa naučiť koncept hypertables, automatickú kompresiu dát, geometrické typy a~priestorové funkcie. Základy relačných databáz zo štúdia mi však poskytli solídny fundament pre rýchle osvojenie týchto rozšírení.

\section{IoT protokoly}
\label{sec:ChybProtokoly}

Špecializované IoT protokoly LoRaWAN~\cite{lorawan_spec} a~MQTT~\cite{mqtt_spec} neboli súčasťou predmetu Počítačové siete. Pri LoRaWAN som sa musel naučiť OTAA aktiváciu a~štruktúru payload správ. MQTT publish/subscribe model som pochopil rýchlo vďaka znalostiam o~sieťovej komunikácii, ale práca s~TLS zabezpečením vyžadovala dodatočné štúdium.

\section{Frontend technológie}
\label{sec:ChybFrontend}

V~oblasti frontendu som mal základné znalosti z~predmetu Vývoj informačných systémov. Počas praxe som ich prehĺbil o~prácu s~knižnicou Leaflet.js~\cite{leaflet_docs} pre mapové vizualizácie a~Socket.IO~\cite{socketio_docs} pre real-time WebSocket komunikáciu.
