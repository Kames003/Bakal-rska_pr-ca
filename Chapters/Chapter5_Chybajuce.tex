\chapter{Znalosti či zručnosti chýbajúce študentovi počas odbornej praxe}
\label{sec:Chybejici}

% Rozsah: 1 strana

Počas odbornej praxe som sa stretol s~niekoľkými technológiami a~nástrojmi, ktoré neboli súčasťou vysokoškolského štúdia. Išlo predovšetkým o~špecializované nástroje pre DevOps, databázové rozšírenia a~IoT protokoly. Vďaka solídnym základom z~predmetov ako Databázové systémy či Počítačové siete som sa však dokázal pomerne rýchlo adaptovať a~chýbajúce znalosti si doplniť štúdiom dokumentácie a~konzultáciami s~mentorom.

\section{DevOps technológie}
\label{sec:ChybDevOps}

Úvodné úlohy, ktoré som dostal pred hlavným projektom, mi umožnili nadobudnúť praktické znalosti o~technológiách, ktoré na fakulte vyučované nie sú, no v~modernom vývoji softvéru sú nevyhnutné. S~Dockerom a~kontajnerizáciou som sa už predtým stretol, ale tieto úlohy prehĺbili moje znalosti a~pripravili ma na ich zakomponovanie do projektu. Išlo konkrétne o~Docker a~Docker Compose pre orchestráciu viacerých služieb, GitLab CI/CD pre automatizované buildy a~deployment, webserver Nginx a~platformu Dokku pre nasadzovanie aplikácií.

\section{Databázové rozšírenia}
\label{sec:ChybDatabazy}

S~rozšíreniami PostgreSQL -- TimescaleDB~\cite{timescaledb_docs} pre prácu s~časovými radami a~PostGIS~\cite{postgis_docs} pre geopriestorové operácie -- som sa počas predmetov Databázové systémy nestretol. Musel som sa naučiť koncept hypertables, automatickú kompresiu historických dát, chunking pre optimalizáciu dotazov, geometrické typy (POINT, POLYGON), priestorové funkcie (ST\_Contains, ST\_Distance, ST\_DWithin) a~GIST indexy pre efektívne priestorové vyhľadávanie. Základy relačných databáz zo štúdia mi však poskytli solídny fundament pre rýchle osvojenie týchto rozšírení. Zároveň som si prehĺbil znalosti s~nástrojmi, s~ktorými som sa počas štúdia stretol len okrajovo -- písanie databázových triggerov v~procedurálnom jazyku PL/pgSQL a~konfigurácia connection poolingu pre efektívnu správu databázových spojení pomocou knižnice psycopg3.

\section{IoT protokoly}
\label{sec:ChybProtokoly}

Špecializované IoT protokoly LoRaWAN~\cite{lorawan_spec} a~MQTT~\cite{mqtt_spec} vysokoškolské osnovy nezahŕňali. Pri LoRaWAN som sa musel naučiť OTAA (Over-The-Air Activation) aktiváciu zariadení, spreading factors ovplyvňujúce dosah a~rýchlosť prenosu, štruktúru payload správ a~ich dekódovanie, konfiguráciu gateway a~network servera, ako aj riešenie problémov s~firmware verziami trackerov. MQTT publish/subscribe model som pochopil rýchlo vďaka znalostiam o~sieťovej komunikácii, ale práca s~TLS zabezpečením, správa certifikátov a~konfigurácia QoS (Quality of Service) úrovní vyžadovala dodatočné štúdium oficiálnej dokumentácie. Praktické skúsenosti som získal aj s~parsovaním payload správ -- dekódovaním hexadecimálnych dát zo SenseCap trackerov a~konverziou surových GPS súradníc do použiteľného formátu.

\section{Frontend technológie}
\label{sec:ChybFrontend}

Frontendovému vývoju som sa venoval už počas štúdia a~vo voľnom čase. Počas praxe som tieto znalosti prehĺbil a~rozšíril o~prácu so špecializovanými knižnicami. Pri Leaflet.js~\cite{leaflet_docs} som sa naučil prácu s~mapovými vrstvami, markermi, polylines pre vykreslenie trás, polygon editovaním pre geofencing a~heatmap pluginom pre vizualizáciu intenzity pobytu. Socket.IO~\cite{socketio_docs} mi prinieslo pochopenie event-based real-time komunikácie, riešenie reconnection scenárov pri výpadku spojenia a~správne poradie spracovania správ. Okrem toho som si prehĺbil znalosti s~technológiami, ktoré som poznal len povrchne -- pokročilé React vzory ako useEffect, useCallback a~useMemo pre optimalizáciu renderingu, utility-first framework Tailwind CSS pre rýchle prototypovanie používateľského rozhrania a~moderný build nástroj Vite pre efektívny development workflow.
