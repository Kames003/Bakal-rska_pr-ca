\chapter{Dosažené výsledky v průběhu odborné praxe a její celkové zhodnocení}
\label{sec:Vysledky}

Počas šiestich mesiacov odbornej praxe som implementoval kompletný GPS tracking systém pokrývajúci celý životný cyklus dát -- od prijatia surových GPS súradníc z~LoRaWAN trackerov až po ich vizualizáciu v~interaktívnej webovej aplikácii. Systém zahŕňa real-time sledovanie pozícií, prehliadanie histórie pohybu, geofencing s~automatickými alertmi a~správu zariadení s~monitorovaním stavu batérie.

\section{Čo by som urobil inak}
\label{sec:CoJinak}

S~odstupom času a~nadobudnutými skúsenosťami identifikujem niekoľko oblastí, kde by som postupoval odlišne. Väčšinu vývoja som realizoval so simulovanými GPS dátami, čo umožnilo rýchly iteratívny vývoj. Avšak niektoré edge cases, ako GPS drift pri slabom signáli alebo battery stuck bug, som objavil až pri testovaní s~reálnymi zariadeniami. Skoršia integrácia reálnych trackerov by tieto problémy odhalila skôr.

Projekt aktuálne nemá formálnu test suite. Implementácia unit testov pre parser a~integration testov pre API by významne uľahčila refactoring a~pridávanie nových funkcionalít. Podobne by implementácia automatizovaného CI/CD pipeline zvýšila kvalitu kódu vďaka automatickým kontrolám pri každom commite.

Retrospektívne písanie dokumentácie pre bakalársku prácu bolo náročnejšie, než by bolo priebežné dokumentovanie rozhodnutí. Odporúčam budúcim študentom viesť si denník vývoja s~poznámkami o~dôležitých rozhodnutiach a~ich zdôvodneniach.

\section{Celkové zhodnotenie}
\label{sec:CelkoveZhodnotenie}

Napriek identifikovaným oblastiam na zlepšenie považujem projekt za úspešný. Podarilo sa mi implementovať funkčný GPS tracking systém, ktorý spĺňa všetky požiadavky zadania a~prináša praktickú hodnotu pre potenciálne využitie na eventoch. Získané skúsenosti s~IoT technológiami, time-series databázami a~real-time webovými aplikáciami sú cenným prínosom pre moju ďalšiu kariéru v~IT.
