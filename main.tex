% === ABSTRAKTY ===
\CzechAbstract{Táto bakalárska práca dokumentuje priebeh odbornej praxe vo firme Railsformers s.r.o., kde som pracoval na vývoji GPS tracking systému pre sledovanie osôb na veľkých podujatiach. Systém využíva technológiu LoRaWAN umožňujúcu komunikáciu na dlhé vzdialenosti pri nízkej spotrebe energie. Práca popisuje návrh a implementáciu kompletného riešenia -- od konfigurácie IoT hardvéru, cez Python backend s PostgreSQL databázou rozšírenou o TimescaleDB a PostGIS, až po React frontend s interaktívnou mapovou vizualizáciou. Výsledný systém umožňuje real-time sledovanie pozícií, zobrazenie histórie pohybu s heatmapami a geofencing s automatickými notifikáciami. Práca reflektuje využité znalosti zo štúdia a identifikuje oblasti, v ktorých bolo potrebné získať nové zručnosti.}

\CzechKeywords{LoRaWAN; GPS tracking; IoT; PostgreSQL; TimescaleDB; PostGIS; Flask; React; MQTT; geofencing; real-time systémy}

\EnglishAbstract{This bachelor thesis documents the professional internship at Railsformers s.r.o., where I worked on developing a GPS tracking system based on LoRaWAN technology. The thesis describes the design and implementation of a complete solution including hardware configuration (RAK7289 gateway, SenseCap T1000-B trackers), Python backend development with Flask framework, database design with PostgreSQL, TimescaleDB and PostGIS, and React frontend with interactive map. The system enables real-time position tracking, historical movement display with heatmaps, and geofencing with automatic notifications. The thesis reflects the knowledge applied from studies and identifies areas where new skills needed to be acquired.}

\EnglishKeywords{LoRaWAN; GPS tracking; IoT; PostgreSQL; TimescaleDB; PostGIS; Flask; React; MQTT; geofencing; real-time systems}
